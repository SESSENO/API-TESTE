Acessos à API
  AddRegistroContabilidade

  ShowRegistros
  GET http://localhost:3000/cont/show
  
  ReportRev
  GET http://localhost:3000/cont/reportRev
  
  ReportExp
  GET http://localhost:3000/cont/reportExp
  
  ShowOp - Despesas
  http://localhost:3000/cont/showop/0
  
  ShowOp - Receitas
  http://localhost:3000/cont/showop/1
